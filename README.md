# FitRechner 🏋️

**Eine kostenlose Sammlung von Fitness- und Ernährungs-Rechnern – auf Deutsch, für GitHub Pages.**

🔗 **Live:** `https://<dein-username>.github.io/fitrechner-de/`

---

## Enthaltene Rechner

### 💪 Kraft & Leistung
| Rechner | Beschreibung |
|--------|-------------|
| [Maximalkraftrechner (1RM)](pages/maximalkraft.html) | Schätze dein 1-Wiederholungsmaximum aus Trainingsgewicht & Wiederholungen (Brzycki, Epley, Lander, O'Conner, Mayhew). Inklusive Trainingsgewichte nach Prozent. |
| [VO₂max Schätzer](pages/vo2max.html) | Maximale Sauerstoffaufnahme schätzen – 4 Methoden: Herzfrequenz, Cooper-Test, Rockport Walking Test, Pace-Methode. |
| [Pace & Durchschnittstempo](pages/pace.html) | Tempo/Pace aus Distanz+Zeit, Zielzeit aus Pace+Distanz, Distanz aus Pace+Zeit. Mit Split-Tabelle für 5k/10k/HM/Marathon. |

### 🔥 Kalorien & Energie
| Rechner | Beschreibung |

## Deployment auf GitHub Pages

1. **Fork oder klone** dieses Repository
2. Gehe zu **Settings → Pages**
3. Wähle **Source: Deploy from branch** → `main` → `/ (root)`
4. Nach wenigen Minuten erreichbar unter `https://<username>.github.io/fitrechner-de/`

Keine Build-Tools, kein npm, keine Abhängigkeiten – reines HTML/CSS/JS.
